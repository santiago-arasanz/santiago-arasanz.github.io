What are the best practices associated with using classes vs. ids?

In HTML and CSS there are lots of selectors. Selectors are used to target a specific element or range of elements on a webpage. Once and element has been targeted, we could apply a style or set of styles.
Today we are going to talk about two selectors: Class and ID.
The ID selector allows us to define style rules that apply to a single element on the webpage. Each webpage can only have one element with a single ID attribute. This means the ID selector can never be used to style more than one element. The Class selector can be used as many times as we want within the same webpage. So, If we have a series of common properties to apply to different elements, it would be appropriate to use class.
